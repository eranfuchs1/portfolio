I love programming. Ever since I wrote my first line of code, I was hooked. My first language was C#, and from there I moved to other languages and frameworks as self taught/hobby (C, C++, Java, Python, Javascript, Bash).




I've had a lot of fun programming the Draw Factory project. In this project you enter a role, which assigns to a particular drawing tool.
Each drawing tool has a position in a conveyor belt so that the first tool gets a blank drawing canvas, the second tool gets what was drawn with the first tool and so on, until all tools have been used in order.
The final product or drawing is displayed in a show-all page.




This project was my first hackathon and first winner of a hackathon. You are a jet fighter pilot, starting off on an aircraft carrier. The map is continously randomly generated, so it's essentially infinite.
With the radar on the bottom right, find and bomb as many enemy fighter jets as possible, which gives your fighter jet extra fuel (fuel can also come from bombing land).
