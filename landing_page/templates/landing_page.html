I've had a lot of fun programming the Draw Factory

project.

In this project you enter a role, which assigns to a

particular drawing tool.

Each drawing tool has a position in a conveyor belt

so that the first tool gets a blank drawing canvas,

the second tool gets what was drawn with the first tool

and so on, until all tools have been used in order.

The final product or drawing is displayed in a show-all

page.
